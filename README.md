## Description
PhotoSpot is a photo-sharing web application where users can create a profile, upload photos of places, like posts, and manage their own content.

## Technologies
- HTML5
- CSS3 (BEM)
- JavaScript (ES6)

## Features
- User profile editing
- Add and delete photo cards
- Like / unlike photos
- Responsive layout
